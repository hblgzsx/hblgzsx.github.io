常用的request对象方法

  1. request.GET 获取url上以？形式的参数
  2. request.POST 获取post提交的数据
  3. request.path ：/index， 打印路径， eg http:/127.0.0.1/index, 那么这个值就是/intex1
  4. request.method 请求的方法 比如：get post
  5. request.COOKIES 请求过来的cookies
  6. request.user 请求的用户对象，可以通过它判断用户是否登录，并获取用户信息。
  7. request.session 一个即可读又可写的类似字典的对象，表示当前会话。
  8. request.META 一个标准的python字典，包含所有的HTTP首部，具体的头部信息取决于客户端和服务器。
